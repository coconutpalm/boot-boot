release

(release & {:as *opts*, :keys [help]})

Release a Jar. If project-type is :open-source, pushes to Clojars. If project-type is :private, pushes to a configured repository. If project-type is :private and no respository is configured, aborts.

For Clojars, depends on CLOJARS_USER, CLOJARS_PASS, CLOJARS_GPG_USER, CLOJARS_GPG_PASS, envars.

Keyword Args: :help bool Print this help info.
